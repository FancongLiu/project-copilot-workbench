# Synthetic design-engineer review brief

Review item `DB-ROLE-01` requires the design engineer to reconcile the current
equipment schedule, approved control sequence, BAS point schedule, and latest
implemented configuration before accepting a performance conclusion.
The synthetic isolation marker for this role document is `DBAUTH5N3`.

The role review must distinguish design intent from measured operation. Its
deliverable is a cited discrepancy list, not a control command. Missing pump,
valve, sensor-quality, or effective-date evidence must be reported as an open
information request rather than filled from memory.
