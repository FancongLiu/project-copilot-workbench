# Synthetic HVAC engineer role data areas

These four data areas are fully synthetic, CC0-1.0 evaluation fixtures. Each
role is imported with the shared Project Aurora package into a separate runtime
workspace. The isolation tests whether retrieval remains grounded in the
engineer's assigned project context; it is not an authorization or identity
system.

The roles cover commercial-HVAC design, controls commissioning, field service,
and project delivery. Defrost replay is one difficult temporal-control case
inside the commissioning role, not the scope of the product.

All equipment, people, identifiers, values, and observations are fictional and
must not be used as engineering instructions for real equipment.
