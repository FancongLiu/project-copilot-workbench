# Synthetic plant configuration

The demonstration plant contains one fictional water-cooled chiller, one primary chilled-water pump, one condenser-water pump, and one cooling tower cell.

The telemetry dataset contains timestamps, supply and return temperatures, electric power, delivered cooling, and plant load percentage. These names and values are synthetic and are not copied from a real facility or product.

The approved efficiency metric is coefficient of performance: cooling output divided by electric power. The approved temperature metric is return-water temperature minus supply-water temperature.
