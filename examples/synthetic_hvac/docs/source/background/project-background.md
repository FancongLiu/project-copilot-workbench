# Project Aurora synthetic cooling-plant upgrade

Project Aurora is a fully fictional commissioning exercise for a small chilled-water plant at the invented Northstar Test Lab. It exists only to test project search, meeting history, configuration reasoning, governed analytics, citations, clarification, and refusal behavior.

The project objective is to demonstrate stable occupied cooling while keeping all software actions read-only. No document in this package authorizes control of physical equipment.

## Equipment register

| P-CHW-01 | Primary chilled-water pump with VFD | 37 kW motor |
| P-CW-01 | Condenser-water pump | 45 kW motor |

| TT-101 | Chilled-water supply temperature sensor | Control feedback |
| TT-102 | Chilled-water return temperature sensor | Performance calculation |

The synthetic acceptance window runs from 2026-06-05 through 2026-07-15. The telemetry file covers 72 hourly observations from 2026-06-30 through 2026-07-02.
