# SOP-SYN-002: synthetic high-temperature review

The fictional high supply-temperature alarm is 7.5 degrees Celsius sustained for 10 minutes.

When reviewing an alarm, first inspect the timestamp, TT-101 supply temperature, TT-102 return temperature, load percentage, and power. Compare the event with the effective configuration and the decision register. Do not infer that a historical baseline is current.

The Workbench may summarize approved records, but it may not acknowledge alarms, change setpoints, start equipment, run shell commands, or browse unrestricted external sources.
